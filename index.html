Our OS Products

Bass OS

Android 12 (Android 14 coming soon)

x86_64-v2/v3 PC's, Laptops, SBC's, Tablets & 2-in-1's 
Bass OS (Broad Apparatus Support System) is a highly configurable open-source toolkit and OS based on Android-x86 and Bliss OS, and it works with most Windows, ChromeOS & Linux PC hardware.

Bass OS contains only proprietary drivers, firmware and codecs, and is Google free by default, but can also come with MicroG included for applications that rely on Google Play Services.   

Bass OS Services

MDM Services

Our Bass application layer allows you the ability to use any MDM compatible with x86_64v2 Android. Contact us if you are interested in this service and we can always help direct you to one of our MDM partners. 

OTA Updates

We can include the ability to update your Bass OS builds via OTA Updates. We also offer the option to self host if needed, or we can host the updates for you.

App Integration

We can integrate your internal applications and pre-grant application permissions for a seamless out-of-the-box experience.

OS Branding

Our Bass Customization layer allows changing the OS name, boot menu, boot animation, wallpaper, and more

Enhanced Security

We offer various Bass addons to provide additional layers of security like locking admin mode access behind a password, or restricting internet access. 

Hardware Interoperability

Bass OS can be customized to allow multiple device configurations in order to allow your company to use one OS image across multiple device targets.

Kiosk mode launcher

Platforms: ARM/ARM64/x86/x86_64

Restricted Launcher is created with added features that allow it to work with our Lockdown & Admin modes built into Bliss OS. It also contains a full set of features for other devices as well. Including its multi-display targeting application auto launch and branding overlay features.
